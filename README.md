# A3: Parallelized Delta-Stepping Algorithm Implementation

This was a project for University of Rochester's "Parallel and Distributed Systems" course. My implementation was parallelized from Professor Scott's sequential implementation of Meyer & Sanders's [Delta-Stepping Algorithm](https://doi.org/10.1016/S0196-6774(03)00076-2). Uses [OpenMPI](https://www.open-mpi.org/doc/v3.1/).

Since the source code is built off a starter file that cannot be shared outside the course, I am keeping the original repository private. However, I am more than happy to talk about my implementation. Please contact me by the email on my resume.
